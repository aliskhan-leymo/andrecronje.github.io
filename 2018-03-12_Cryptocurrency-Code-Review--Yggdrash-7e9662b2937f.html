Cryptocurrency Code Review: Yggdrash

Disclaimer: These reviews are done as is from what is on display in the master branch of the repo’s made available. This review is not a comment on the overall project, scope, or success thereof. This was done as an educational review by me and any comments in the article are simply my opinion. It should not be used as any comment or advice on the project as a whole.

“YGGDRASH is a trust-based multi-dimensional blockchains (branches) built with a vision “Digitize everything into reality” and to connect everything and any blockchain networks.

YGGDRASH’s main-net is called STEM, and its cryptocurrency is YEED.
It is a trust-based blockchain ecosystem where each branch is governed by reputation to ensure integrity.”

We are having a look at their POC

“yggrash Proof of concept, just Mockup”, so this isn’t actually a proof of concept, it’s a mockup of a proof of concept, so we aren’t expecting much.

Fake blockchain data for the fake PoC.

Fake assets mapping for the fake PoC.

Fake conversion metrics for the fake PoC.

HTTP (insecure data transfer) instead of HTTPS, really guys, a self-signed cert takes 2 minutes.

File lookups everytime we want to display data, again guys, a Redis instance takes 2 minutes, keep that data in memory. Express (The node.js API software they are using) even allows you to autocache them in routes.

Nothing else to see here.

Conclusion: This is stock standard boiler plate for an express/jade/node.js server (in fact some of the boiler plate is still in), it doesn’t show what they can do unfortunately since anyone can deploy something similar, so can’t deduce much about the teams skillset if anything about this.

What does speak volumes is that this repo was started on the 17th of Jan and was only completed on the 11th of March. That’s a very long time for something that should have been a weekend worth of work.

They definitely should not be calling a mockup a PoC though, that’s blatantly lying to their users. Will wait untill they publish some more concrete code.
