Cryptocurrency: Elastos, NKN, DNA Project and code credit?

Disclaimer: These reviews are done as is from what is on display in the master branch of the repo’s made available. This review is not a comment on the overall project, scope, or success thereof. This was done as an educational review by me and any comments in the article are simply my opinion. It should not be used as any comment or advice on the project as a whole.

EDIT 23/03/2018, Kiran Pachhai correctly notes that this article does not provide a lot of evidence, as such I will be adding the additional evidence in this edit.

Let’s start with the import ordering (first Elastos, second DNA project)

Note the same ordering, and package names, at this point already a developer would tell you the chances of having the same ordering is unlikely, but let’s move on

Same variable names, and spacing, he however notes that initialization is fairly textbook, so they could have coppied from the same source, let's move on

Same variable names, same variable layout, both have a protocol called Noder

Same variables and same hardcoded values for variables, even the same comments.

I could keep on, but at this point you can see the pattern repeat for yourself in the rest of the codebase. I’m just asking, why wasn’t credit given to the original system? Why is there no mention of DNA project? Why was the original commit log wiped? I believe source should be coppied and improved, but I believe credit should be given where it is due.
