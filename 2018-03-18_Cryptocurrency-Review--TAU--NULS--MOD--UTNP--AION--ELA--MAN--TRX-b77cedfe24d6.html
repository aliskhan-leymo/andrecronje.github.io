Cryptocurrency Review: TAU, NULS, MOD, UTNP, AION, ELA, MAN, TRX

Disclaimer: These reviews are done as is from what is on display in the master branch of the repo’s made available. This review is not a comment on the overall project, scope, or success thereof. This was done as an educational review by me and any comments in the article are simply my opinion. It should not be used as any comment or advice on the project as a whole.

Expanded the existing model again a little bit, still tweaking it for optimum results, the current aggregates used can be displayed as follow;

Social { daily, weekly, monthly} activity (likes/upvotes/retweets/new telegram users)
Social { daily, weekly, monthly} posts
Social followers { facebook, twitter, linkedIN, telegram, reddit }
Concept { type, market, competition, … }
Technical team { Skill rankings, Skill relevance, Recommendations, Recommendation relevance, experience years }
Advisor team { Skill rankings, Skill relevance, Recommendations, Recommendation relevance, experience years }
Prototype { key features, completion, value proposition }
[GitHub { commits, contributors, cross commits, forks, watchers }]
Token { total, circulating, price, circulating delta }
Sector { type, competitor max cmc, competitor ROI }
Category { type, competitor max cmc, competitor ROI }

I started this research based off of Piotr Plonski’s (https://medium.com/@MLJARofficial/predicting-ico-returns-with-machine-learning-af6108ab9e39) machine learning ICO prediction model, so credit to him for the original idea. The difference is that he used the outcome prediction as one of the inputs, I’m using the model to predict the value as an output.

I will do a full article on this soon, for now the results;

Lamden

Lamden’s 73% triggered my interests so I did a full review here https://medium.com/@suchi.blackwing/cryptocurrency-code-review-lamden-tau-c973669c2219 their code unfortunately was very junior and unimpressive, I don’t see how they will be able to compete in this competitive market.

AION

My AION review I started some time ago, I will still release this. I am keeping a close eye on them

Elastos

While Elastos didn’t score great, their github intrigued me, I did a full review here https://medium.com/@suchi.blackwing/cryptocurrency-code-review-elastos-ela-1b316b5a5607 I ended up really not liking their code and it lacks a lot of optimization, the thing that got me the most is that it is just blockchain 101 code, nothing interesting here.

Matrix

Matrix scored high, so I was interested to have a look at their code;

Lots of repo’s (16 in total), but very little of substance. The only project that has received recent commits is the Whitepaper. So nothing to review here currently. Untill they release something of substance I will be ignoring this.

Some of the repo’s that they do have are a bit troubling, for example Deep-Leaning-Tutorial and Hello-World, these are beginner tutorials and they were recently (August 2017) created, so perhaps not the most senior team.

Tron

Didn’t score great, but enough people have asked me to review it, so I gave it a go https://medium.com/@suchi.blackwing/cryptocurrency-code-review-tron-trx-2c9565772cb1 it’s high caliber code, but they are very far behind the rest in terms of progress, nowadays you have to be faster and innovating, they are still trying to catch up. Their partnerships might give them an edge here, but code wise they are quite far behind.

Conclusion: I am liking the results of this model so far, it seems to be getting quite accurate. I need to sit down one day and run all of the current top 100 through it and see the results.
